ระบบบัตรเงินสด อัปเดตล่าสุดเมื่อ: วันอังคารที่ 22 ธันวาคม พ.ศ.2563

ข้อมูลเบื้องต้น

ระบบนี้ถูกพัฒนาเป็นครั้งแรกเมื่อวันพุธที่ 2 ธันวาคม 2563 โดยได้รับการว่าจ้างจากบริษัทหนึ่งซึ่งได้ถามไถ่และตกลงกับทางบริษัทดังกล่าวแล้วว่าผมผู้พัฒนานั้น สามารถนำระบบนี้มาต่อยอดและใช้งานได้ดังนั้นแล้วจึงได้เกิดเป็นระบบบัตรเงินสดนี้ขึ้นมา

สิ่งที่จำเป็นต่อการใช้งาน:

  • ติดตั้ง NodeJS เวอร์ชั่น 14.15 ขึ้นไป (โหลดได้ที่ เว็บไซต์ NodeJS)
  • ระบบฐานข้อมูล เช่น MySQL, MariaDB (แนะนำให้ใช้ MariaDB)
  • สร้าง RSA KeyPair ขนาดมากกว่าหรือเท่ากับ 256-bits (ประกอบด้วย Private Key และ Public Key)

ขั้นตอนการขอใช้งาน:

  1. เข้าไปที่เว็บไซต์ เริ่มต้นใช้งานบัตรเงินสด
    *หากยังไม่เคยสมัครกรุณาสมัครสมาชิกก่อน
  2. หลังจากได้รับคีย์และไฟล์ระบบมาแล้ว, ให้แตกไฟล์ระบบไว้ในเซิฟเวอร์ (ข้ามขั้นตอน 2-5 และติดตั้งที่นี่ หากใช้งานระบบออนไลน์)
  3. คัดลอกไฟล์ private.key และ public.pub มาไว้ในโฟลเดอร์ includes (ชื่อต้องถูกต้อง)
  4. ติดตั้งโมดูลของ NodeJS ตามที่ให้ไว้ดังนี้:
    • Express.js: npm install express
    • Morgan.js: npm install morgan
    • Express.js Handlebars: npm install express-handlebars
    • Body Parser: npm install body-parser
    • Cookie Parser: npm install cookie-parser
    • Cookie Encrypter: npm install cookie-encrypter
    • JSONWebToken: npm install jsonwebtoken
    • Edit JSON File: npm install edit-json-file
    • MySQL: npm install mysql
    • ติดตั้งทั้งหมด: npm install express express-handlebars morgan body-parser cookie-parser cookie-encrypter jsonwebtoken edit-json-file mysql
  5. สั่ง npm ./bin/www เพื่อเริ่มใช้งาน และเข้า http://˂YOUR-HOSTNAME˃ เพื่อเข้าสู้หน้าตั้งค่าเบื้องต้น
  6. การติดตั้งเสร็จสมบูรณ์

การติดตั้ง Modules

Modules เป็นส่วนเสริมที่มีรูปแบบการติดตั้งเป็น JSON files แบบเข้ารหัสจำเพาะโดยมีวิธีการติดตั้งอยู่ 3 วิธีด้วยกัน

1 - ติดตั้งผ่านทางระบบติดตั้งออนไลน์

การติดตั้งวิธีนี้จำเป็นต้องอัปโหลดระบบและต้องใช้โดเมนที่ลงทะเบียนไว้


ขั้นตอนการติดตั้ง:
  1. เข้าไปที่แผงควบคุมของผู้ดูแลระบบ
  2. ไปที่ (การตั้งค่าส่วนเสริม) แล้วกดปุ่ม [+ ติดตั้งส่วนเสริม]
  3. เลือกแถบ ดาวน์โหลดออนไลน์
  4. ล็อกอินด้วยบัญชี xAccount (หากระบบตรวจสอบ APIKey แล้วผ่านจะไม่เจอขั้นตอนนี้)
  5. เลือกส่วนเสริมที่ต้องการแล้วกด [+ ติดตั้งส่วนเสริม]
